## Where to Start

**If you're new to Linux:**
→ Start at [Roadmap — Phase 1](./roadmap/README.md#phase-1--getting-comfortable-with-linux)

**If you want to look something up:**
→ Go to the [Bible](./bible/README.md) and find the chapter

**If you want a lesson on a topic:**
→ Use the `teach-linux:teach` skill — type the topic, get a full deep-dive

**If you want to test your knowledge:**
→ Use the `teach-linux:test` skill
